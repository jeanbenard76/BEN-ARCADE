GUIDE OFFICIEL BEN'ARCADE

À propos du décompte

Cette application est conçue pour simplifier vos soirées jeux. Les calculs sont automatisés selon les règles standards. En cas de litige, référez-vous aux détails ci-dessous. Les options de configuration avancées arrivent prochainement !

Tarot

•

Le Tarot se joue à 3, 4 ou 5 joueurs.

•

Points de contrat : Petite (x1), Garde (x2), Garde Sans (x4), Garde Contre (x6).

•

Bouts : Le Petit, le 21 et l'Excuse. Ils réduisent le nombre de points nécessaires pour gagner.

•

0 bout : 56 pts, 1 bout : 51 pts, 2 bouts : 41 pts, 3 bouts : 36 pts.

•

Primes : Petit au bout (10 pts x contrat), Poignée (Simple 20, Double 30, Triple 40), Misère (10 pts), Chelem (200/400 pts).

Configuration Standard

Skull King

•

Chaque joueur parie sur le nombre de plis qu'il va réaliser.

•

Pari réussi : 20 pts par pli (ou 10 pts x manche si pari à 0).

•

Pari raté : -10 pts par pli de différence (ou -10 pts x manche si pari à 0).

•

Bonuses : Cartes 14 (+10), Sirène bat Pirate (+20), Pirate bat Skull King (+30), Skull King bat Sirène (+40).

Configuration Standard

Flip 7

•

Le but est d'accumuler le plus de points sans retourner deux fois le même chiffre.

•

Si vous retournez un 7, vous perdez tout pour cette manche.

•

Bonus Flip 7 : Si vous parvenez à retourner 7 cartes différentes, vous gagnez un bonus de 15 points.

Configuration Standard

Skyjo

•

Le but est d'avoir le moins de points possible à la fin de la partie.

•

La partie s'arrête quand un joueur atteint 100 points.

•

Si vous alignez 3 cartes identiques en colonne, elles sont défaussées (0 point).

•

Le joueur qui termine la manche doit avoir le score le plus bas, sinon son score est doublé.
